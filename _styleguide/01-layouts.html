--- title: Layouts ---

Single Column

A single column is mostly used for text content, such as diary posts.

.l-column

Push and Pull

Push and Pull are utility classes that are used to break content out of the single column. These are typically used for media elements in diary posts.

Push

.push-left

.push-left is used to align items to the left of inline elements, at wider breakpoints.

.push-right

.push-right is used to align items to the right of elements, at wider breakpoints.

Pull

.pull-left

.pull-left is used to pull items out of column to the left of inline items, at wider breakpoints.

.pull-right

.pull-right is used to pull items out of column to the right of inline items, at wider breakpoints.

.pull-both

.pull-both is used to pull block level elements into a wider column on both sides. Usually used for images and galleries within posts.

.pull-extended

.pull-extended is used to pull block level elements into a max-width column on both sides. Usually used for galleries within gallery pages.

Repel

Repel is used to add vertical space under elements.

.repel-none

.repel-none is used to turn off repel between a block level element and elements below.

.repel-1

.repel-1 is used on a block level element to to repel elements below by a factor of 1. (1 x line height spacing).

.repel-2

.repel-2 is used on a block level element to to repel elements below by a factor of 2. (2 x line height spacing).

.repel-3

.repel-3 is used on a block level element to to repel elements below by a factor of 3. (3 x line height spacing).
