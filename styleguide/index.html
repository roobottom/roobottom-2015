Styleguide

All the content within this page is contained with the .prose class scope. This is used when writing blocks of text.

Headings

Second level heading

Second level headings h2 are used in a post. Multiple second level headings can be used to introduce separate ideas in a post.

Third level heading

Third level headings h3 are used in a post as a sub level heading to a h2.

Fourth level heading

Fourth level headings h4 are used in a post as a sub level heading to a h3.

Grouping Content

Horizontal rule

The horizontal rule is used to separate one concept from another in a post.

Usually, then, followed by a second level heading

Blockquotes

The blockquote element is used when quoting from another source

The history of the Olympics is not only one of remarkable sporting achievement, but acts of incredible artistry too.

Ordered lists

Ordered lists ol are used, as their name suggests, to list items which have an order. They can be nested.

  1. UX tasks
    1. User research
    2. Creating personas
    3. Story writing
    4. Sketching
    5. Wireframes
  2. Design tasks
    1. Sketching
    2. Screen mockups
    3. Pattern systems
    4. Screen flows

Unordered lists

Unordered lists ul are used, as their name suggests, to list items which don't have an order. They can be nested.

    • Red
    • Yellow

    • Green
  • Textures

Text level semantics

Links

The a tag is used to link text to another location on the web.

Emphasis

The em tag is used to add emphasis to a word or phrase.

Importance

The strong tag is used to add importance to a word or phrase.

Media

Image

Most new posts will use the figure tag to wrap an image, however historical posts will use the img tag directly.

figures

A figure tag is used to present a single image. It can have an optional caption with figcaption.

Gallery

A gallery component is used when two or more images need to be contained together. It uses javascript to achieve the 'brick wall' effect.

Layouts

There are several utility classes that can be used to either push or pull content, typically media elements, to one side or another.

Push

.push--left.push--left is used to align items to the left of inline elements.

.push--right.push--right is used to align items to the right of elements.

Pull

.pull--left.pull--left is used to pull items to the left of inline items

.pull--right.pull--right is used to pull items to the left of inline items

.pull--both

.pull--both is used to pull block level elements into a wider column on both sides. Usually used for images and galleries within posts.
